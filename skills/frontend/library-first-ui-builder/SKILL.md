# Library-First UI Builder

Build library-first and custom-composed. Use primitives -> wrappers -> sections -> pages. Use Figma/source design when available. Use shadcn/Radix for primitives. Use local wrappers for brand style. Treat section libraries as supply, not final mismatched design. Normalize external blocks into the project design system. Verify rendered results when visual.
