# General Writing Stylecard

## Core Rules

Write clearly.
Use natural sentence flow.
Prefer specific details over generic claims.
Avoid robotic transitions.
Avoid over-polished marketing language unless the selected profile requires marketing copy.
Avoid repeating the same sentence pattern.
Avoid vague claims unless they are followed by specific explanation.
Use direct verbs.
Keep paragraphs focused.
Vary sentence length.
Keep technical claims accurate.
Do not invent facts, dates, tools, features, results, sources, or project work.
Match the writing style to the selected profile.
Make the writing sound like a real person explaining real work, not a template.

## Preferred Writing Behavior

Use simple but mature wording.
Use examples when they help.
Use active voice where natural.
Use first person only when the profile allows reflection or personal experience.
Use evidence-based wording for formal documents.
Use practical wording for technical documents.
Use benefit-focused wording for website and sales copy.
Use concise wording for emails and proposals.
