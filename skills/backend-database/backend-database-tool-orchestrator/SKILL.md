# Backend Database Tool Orchestrator

Route unknown projects to project scan; API work to API contract; DB work to database safety; raw SQL to SQL operations; migrations/backups/restores to migration gate; SSH/VPS to VPS gate; logs to observability; deployments to readiness; env/security to security gate; performance to performance triage; refactors to refactor safety.

Core rule: read-only discovery first, then smallest safe change, then verification.
