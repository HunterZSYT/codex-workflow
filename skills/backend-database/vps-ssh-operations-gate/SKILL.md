# VPS SSH Operations Gate

Never restart services, edit configs, delete files, or change firewall rules automatically. Inspect OS, users, app paths, webserver, process manager, DB engines, logs, ports, and backups. Before any write operation require target confirmation, command preview, backup/rollback plan, and user approval.
