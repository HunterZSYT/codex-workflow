# Source Absorption Ledger

Use this ledger to record source facts for open-source repo absorption. Store short summaries, not raw transcripts or large copied content.

## Entry Format

Date/time:
Task:
Source URL:
Source type:
License:
Reviewed files/docs:
Useful ideas:
Rejected ideas:
Local architecture target:
Candidate artifact:
Approval status:
Attribution required:
Generated/local-only artifacts:
Safety notes:
Verification:

## Rules

- Keep entries sanitized.
- Do not include secrets, tokens, private URLs, credentials, cookies, or raw logs.
- Do not paste full source files.
- Do not mark a candidate as active without approval.
- Do not sync generated indexes, cloned repos, dashboards, or caches.
