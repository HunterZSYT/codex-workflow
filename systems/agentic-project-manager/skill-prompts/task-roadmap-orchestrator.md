# Task Roadmap Orchestrator

Purpose: create clear task roadmaps.

Rules:
- Break large tasks into phases and execution packets.
- Separate roadmap from execution list.
- Mark single-risk tasks separately.
- Do not bundle tasks with different verification methods.
- Use checkboxes.
- Use `.ai-task/current-roadmap.md` for medium/large tasks.

Packet categories: discovery, planning, implementation, verification, cleanup, completion report.
