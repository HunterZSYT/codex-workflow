# Document Parsing Stack

Active decision pack for source-to-knowledgebase ingestion using broad converters, local PDF parsers, and OCR backends.

LiteParse contributes local spatial PDF parsing and OCR-pipeline ideas. PaddleOCR contributes heavy optional OCR/document parsing backend guidance. MarkItDown remains the broad file-to-Markdown baseline when available.

## Safe To Use Now

- Parser/OCR decision tree.
- Trust boundary for extracted text.
- Sync-ignore rules for generated/private ingestion artifacts.

## Approval-Gated

- Installing LiteParse.
- Installing PaddleOCR.
- Downloading OCR models.
- Running OCR on private documents.
- Enabling OCR HTTP servers or services.
