# Examples

- Text-based DOCX/PDF to Markdown: use broad converter first.
- Digital PDF with bounding boxes needed: consider LiteParse after approval if not installed.
- Scanned document: OCR needed; ask approval for private docs.
- Complex forms/tables/multilingual OCR: consider PaddleOCR after install/model review.
- Critical report reconstruction: verify extracted text manually against the PDF.
