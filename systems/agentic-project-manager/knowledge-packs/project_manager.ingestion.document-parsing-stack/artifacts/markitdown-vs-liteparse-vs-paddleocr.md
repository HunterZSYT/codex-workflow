# MarkItDown vs LiteParse vs PaddleOCR

| Tool class | Best for | Boundary |
| --- | --- | --- |
| MarkItDown-style converter | broad file-to-Markdown conversion | may lose layout/spatial detail |
| LiteParse | local PDF spatial parsing, bounding boxes, local OCR pipeline candidate | install approval required |
| PaddleOCR | OCR backend, document layout/table/structure parsing, multilingual OCR | heavy install/model/private OCR approval required |
| Cloud/VLM parser | very hard documents | paid/API/privacy approval required |
