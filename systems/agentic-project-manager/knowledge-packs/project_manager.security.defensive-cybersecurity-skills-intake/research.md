# Research

Anthropic-Cybersecurity-Skills is a large Apache-2.0 cybersecurity skill library. The official README reports 754 skills, 26 domains, agentskills.io compatibility, and mapping to five frameworks: MITRE ATT&CK, NIST CSF 2.0, MITRE ATLAS, MITRE D3FEND, and NIST AI RMF.

The library is not safe to import wholesale. Its domains include defensive uses such as incident response, threat hunting, cloud security, compliance, secure review, and forensics, but also operational areas such as malware analysis, exploit detection, penetration testing, red-team material, and procedures that can become harmful if imported without filtering.

Safe absorption is the defensive intake model and framework mapping pattern, not the raw skills.
