# Examples

## Safe

- Map a secure coding review to NIST CSF categories.
- Add a defensive checklist for secret leakage review.
- Use MITRE/NIST/D3FEND names to organize findings.

## Rejected Or Approval-Gated

- Import a malware-analysis procedure wholesale.
- Run exploit tooling.
- Use offensive red-team playbooks operationally.
- Analyze private security logs without explicit approval and redaction plan.
