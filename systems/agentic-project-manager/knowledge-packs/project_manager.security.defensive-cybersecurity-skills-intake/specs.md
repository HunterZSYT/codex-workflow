# Specs

## Intake Filter

Allow:

- defensive detection logic
- secure coding review checklists
- incident-response triage structure
- cloud/configuration safety checks
- framework mapping and taxonomy
- AI security risk review

Restrict or reject:

- exploit execution
- malware operation
- credential theft
- persistence
- evasion
- privilege escalation execution
- phishing execution
- lateral movement playbooks
- payloads, command recipes, or operational offensive procedures

## Manual Approval Required

Ask before installing security tools, running security scans against real targets, ingesting logs with sensitive data, executing any security command, or importing external cybersecurity skills.
