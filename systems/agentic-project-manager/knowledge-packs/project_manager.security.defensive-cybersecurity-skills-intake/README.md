# Defensive Cybersecurity Skills Intake

Active intake policy for safely learning from cybersecurity skill libraries without importing harmful operational procedures.

This pack uses the Anthropic-Cybersecurity-Skills repo as a source for framework mapping and defensive checklist structure. It does not import the skill library.

## Safe To Use Now

- Defensive-only intake filter.
- Framework mapping notes.
- Secure coding and AI security risk review checklists.

## Rejected Or Restricted

- Offensive exploit playbooks.
- Malware operation procedures.
- Credential theft, persistence, evasion, phishing, or lateral movement execution steps.
- Tool outputs/logs from security operations unless sanitized and explicitly approved.
