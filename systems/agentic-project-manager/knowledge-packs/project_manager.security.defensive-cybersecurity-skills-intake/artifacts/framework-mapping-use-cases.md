# Framework Mapping Use Cases

Use framework names to organize defensive work:

- MITRE ATT&CK: adversary behavior taxonomy for detection coverage.
- NIST CSF 2.0: governance, identify, protect, detect, respond, recover.
- MITRE ATLAS: AI/ML adversarial threat taxonomy.
- MITRE D3FEND: defensive countermeasure vocabulary.
- NIST AI RMF: AI risk governance, mapping, measurement, and management.
