# Cyber Skill Safety Filter

Before using external cybersecurity skill material, classify:

- Defensive detection or hardening: can be summarized.
- Secure review checklist: can be adapted.
- Forensics triage: approval-gated if private data/logs are involved.
- Offensive operation: reject or restrict.
- Tool command: approval-gated.
- Payload/script/procedure: do not import.
