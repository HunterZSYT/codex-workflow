# last30days When To Skip

Skip last30days, or use it only as secondary signal, for:

- official API behavior
- exact installation commands unless official docs confirm
- security-sensitive tasks
- private/internal information
- legal, medical, financial, production, auth, payment, or database decisions
- tasks where old stable docs are enough
- tasks where social signal would add noise
- scraping restricted/private data
- bypassing paywalls, auth, or ToS
