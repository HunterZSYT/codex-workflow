# Source Trust Ladder

Highest trust for implementation:

1. Official docs, standards, and specifications.
2. Official repositories, release notes, changelogs, and source code.
3. Package registries and signed/distributed package metadata.
4. Issue discussions, pull requests, and GitHub activity.
5. Expert posts with clear identity and reproducible evidence.
6. Reddit, Hacker News, Discord/forum/community sentiment.
7. TikTok, X/Twitter, Instagram, Threads, Pinterest virality.
8. Prediction markets.
9. Generic blogs and newsletters.
10. Unknown or unattributed sources.

Rule: social signal is evidence of attention, pain, opinion, or adoption. It is not proof of correctness.
