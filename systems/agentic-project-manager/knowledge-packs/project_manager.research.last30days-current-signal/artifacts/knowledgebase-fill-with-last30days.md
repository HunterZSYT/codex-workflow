# Knowledgebase Fill With last30days

Workflow:

1. User asks for a topic, tool, repo, ecosystem, or current source discovery.
2. Run local retrieval first.
3. Run last30days for current signal when recency/community evidence matters.
4. Save raw output to a local-only directory.
5. Verify durable facts with official docs, official repos, package registries, and Context7 where applicable.
6. Create or update source ledger entries with links and short summaries.
7. Create or update candidate pack/blob/artifacts.
8. Do not activate weak findings automatically.
9. Use Knowledge Sufficiency Gate before implementation.
