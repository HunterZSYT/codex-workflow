# Normal Browser vs DevTools vs Playwright

- Normal browser: manual inspection, public pages, authenticated user-controlled browsing when the user explicitly provides context.
- Chrome DevTools/in-app Browser: local app debugging, DOM inspection, console/network checks, screenshots.
- Playwright: repeatable local/owned UI checks, smoke tests, deterministic interactions.
- Restricted browser: any anti-detect, stealth, proxy-profile, or bot-detection-specific browser. Use only after approval.
