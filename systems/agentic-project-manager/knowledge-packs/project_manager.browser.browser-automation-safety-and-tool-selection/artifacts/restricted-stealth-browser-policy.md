# Restricted Stealth Browser Policy

Stealth and anti-detect browser tools are not normal inspection tools.

Approval must cover:

- target and authorization
- whether account/session/cookies/proxies are involved
- artifact paths and cleanup
- why normal browser, DevTools, Playwright, or web search are insufficient
- what will not be attempted

Never use restricted browser tooling to bypass access controls, CAPTCHA, rate limits, or bot-detection protections on third-party services.
