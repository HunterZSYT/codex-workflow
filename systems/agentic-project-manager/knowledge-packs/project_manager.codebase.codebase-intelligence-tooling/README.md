# Codebase Intelligence Tooling

Active routing pack for choosing between direct file reads, project scans, CodeGraph, existing Understand skills, and Understand Anything-style knowledge graph workflows.

The pack absorbs safe concepts from Understand Anything, not the external install.

## Safe To Use Now

- Tool-selection rules for codebase intelligence.
- Generated graph local-only policy.
- Architecture understanding workflow.

## Approval-Gated

- Installing Understand Anything.
- Running heavy graph generation on a real project when not already requested.
- Syncing `.understand-anything/` generated graph folders.
