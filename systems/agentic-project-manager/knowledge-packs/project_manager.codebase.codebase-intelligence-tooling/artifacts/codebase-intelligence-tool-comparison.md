# Codebase Intelligence Tool Comparison

| Need | Preferred tool |
| --- | --- |
| Exact symbol lookup | CodeGraph or rg |
| Callers/callees/impact/path tracing | CodeGraph |
| Small known-file edit | rg/read direct files |
| Unknown project setup | project capability scan |
| Architecture explanation | Understand Anything / Understand skills |
| Domain/business flow | Understand domain workflow |
| Onboarding guide | Understand onboarding workflow |
| Huge research/source material | Headroom analysis on sanitized local-only bundle |
