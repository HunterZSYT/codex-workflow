# Research

Understand Anything overlaps with the user's existing Understand skills and CodeGraph-style routing, but it adds a useful conceptual model: generate an explorable knowledge graph for architecture learning, business-domain understanding, guided tours, semantic search, and diff impact review.

The README explicitly mentions `.understand-anything/knowledge-graph.json`, dashboard exploration, domain extraction, and generated scratch files. It also suggests committing graph outputs in some team settings. Codex Workflow should be stricter: generated graphs are local-only by default and synced only after explicit approval.

Current signal supports interest in local-first graph/context engines, but official README/docs remain technical truth.
