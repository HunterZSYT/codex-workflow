# Verification

## Pack Verification

This pack is verified when:

- Knowledge search returns the pack for layer/scope queries.
- Search returns new blobs for interaction, responsive, state, media, forms, and navigation queries.
- Sufficiency checks identify layer/scope mapping for representative frontend tasks.
- Pack audit finds source ledger, specs, artifacts, and verification docs.
- Pack remains candidate until approval.

## Task Verification

Use the smallest proof that matches the affected layer:

- Copy/content-only: code/text inspection.
- Layout/spacing/type/color visual change: rendered proof when claiming visual success.
- Mobile/responsive: mobile emulation.
- Overflow/sticky/nav: rendered inspection plus DOM/CSS measurement when useful.
- Interaction states: hover/focus/active/disabled/loading proof when changed.
- Forms/accessibility/nav: keyboard/focus/semantic review and axe when available.
- Performance/media: Lighthouse/performance check only when performance is in scope.
