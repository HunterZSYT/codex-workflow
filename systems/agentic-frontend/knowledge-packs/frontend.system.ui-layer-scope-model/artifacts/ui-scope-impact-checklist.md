# UI Scope Impact Checklist

## 1. Smallest Affected Scope

- [ ] Token
- [ ] Element
- [ ] Primitive component
- [ ] Component group
- [ ] Section
- [ ] Page
- [ ] Template
- [ ] Site/system

## 2. Higher-Scope Impact

- [ ] Button padding affects component primitive?
- [ ] Card spacing affects section rhythm?
- [ ] Hero typography affects page hierarchy?
- [ ] Navigation state affects whole-site interaction language?
- [ ] Mobile header issue affects navigation and responsive structure?

## 3. Local or Systemic Fix

- [ ] Local one-off fix
- [ ] Update component primitive
- [ ] Update section pattern
- [ ] Update page template
- [ ] Update design tokens/system

## 4. What Not To Change

- [ ] Content order
- [ ] Design direction
- [ ] Unrelated sections
- [ ] Active system tokens
- [ ] Unrelated components

## 5. Proof Needed

- [ ] No proof for docs-only
- [ ] Code inspection only
- [ ] Screenshot
- [ ] Mobile emulation
- [ ] DOM/CSS measurement
- [ ] Accessibility check
- [ ] Performance check
- [ ] Interaction state proof
