# Examples

## Good Fit

User asks:

> Make this creative studio homepage feel more editorial and precise.

Use this candidate pack to plan a grid-first composition:

- large left-aligned headline
- small metadata rail
- image aligned to grid columns
- restrained accent color
- strong negative space
- mobile single-column reading order
