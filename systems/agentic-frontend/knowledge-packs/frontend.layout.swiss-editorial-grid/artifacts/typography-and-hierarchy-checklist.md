# Typography and Hierarchy Checklist

Before implementing:

- Identify the one primary message.
- Decide whether typography or image is the focal point.
- Choose one primary sans-serif family unless brand rules require otherwise.
- Use no more than 3 to 5 meaningful type sizes in one section.
- Keep body copy readable and width-controlled.

Swiss/editorial hierarchy:

- Primary headline: large, left-aligned, tight line-height.
- Supporting copy: smaller, readable line-height.
- Metadata: small and structured; uppercase only with letter spacing.
- CTA: aligned to the content block it concludes.

Avoid:

- Centering all type by default.
- Using many font weights to fake hierarchy.
- Using all caps for long text.
- Letting oversized display type create unreadable mobile wraps.
- Making typography decorative instead of structural.

Verification:

- The main message is obvious in 3 seconds.
- Supporting text does not compete with the headline.
- Metadata provides orientation.
- Mobile heading wraps cleanly.
