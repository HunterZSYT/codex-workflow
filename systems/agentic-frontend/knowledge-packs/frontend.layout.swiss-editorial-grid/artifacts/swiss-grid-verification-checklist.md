# Swiss Grid Verification Checklist

Use this before claiming a Swiss/editorial layout is correct.

## Layout

- Shared container is visible across header/section/content.
- Grid lines explain placement.
- Asymmetry feels balanced, not accidental.
- Negative space creates focus.
- No element floats without belonging to a group.

## Typography

- Primary text is dominant.
- Body text is readable.
- Metadata is useful and subdued.
- No random type sizes or weights.

## Visual Restraint

- Color is restrained and purposeful.
- Imagery is content-bearing.
- Decorative shapes are absent or clearly justified.
- The section does not look like a generic card grid.

## Responsive

- Desktop columns collapse logically.
- Mobile keeps source order.
- No text overlap.
- No horizontal overflow.
- CTA remains connected to its content group.

## Source Safety

- No copied poster layout.
- No copied museum images.
- No source asset reuse.
- Public references are used as principles only.
