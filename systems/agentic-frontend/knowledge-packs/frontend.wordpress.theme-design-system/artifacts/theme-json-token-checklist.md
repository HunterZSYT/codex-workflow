# theme.json Token Checklist

- [ ] Palette names are semantic and reusable.
- [ ] Font sizes follow the project type scale.
- [ ] Font families map to real loaded fonts.
- [ ] Spacing presets reflect the spacing rhythm.
- [ ] Layout `contentSize` and `wideSize` match the page container system.
- [ ] Block gap is intentionally controlled.
- [ ] Global element styles are set before custom CSS.
- [ ] Block-level overrides are used only where needed.
- [ ] Style variations are separate JSON files, not one-off CSS hacks.
