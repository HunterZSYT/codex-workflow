# WordPress Layout / Spacing Map

- Site max content width -> `settings.layout.contentSize`
- Wide sections -> `settings.layout.wideSize`
- Block gap -> `settings.spacing.blockGap`
- Section rhythm -> pattern composition and block spacing
- Container alignment -> template and pattern structure
- Inner component padding -> block style or custom class when needed
- Repeated spacing values -> theme.json spacing presets

Avoid random per-block spacing overrides when a reusable preset or pattern rule can express the relationship.
