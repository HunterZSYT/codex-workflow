# Verification

For real WordPress design-system work:

- Confirm `theme.json` parses.
- Confirm color, typography, spacing, and layout tokens appear in the editor as intended.
- Confirm front-end output matches editor preview closely.
- Confirm block patterns render with expected spacing, typography, and responsive behavior.
- Confirm template parts preserve container alignment.
- Use rendered browser inspection for visual/layout/mobile claims.
- Check accessibility contrast and reduced editor misuse.
