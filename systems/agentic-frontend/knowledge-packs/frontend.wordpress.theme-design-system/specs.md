# Specs

Status: candidate.

## Token Mapping

- Color palette: `settings.color.palette`
- Typography presets: `settings.typography.fontSizes`, `settings.typography.fontFamilies`
- Spacing: `settings.spacing.spacingSizes`, `settings.spacing.units`, `settings.spacing.blockGap`
- Layout width: `settings.layout.contentSize`, `settings.layout.wideSize`
- Global styles: `styles`
- Element styles: `styles.elements`
- Block styles: `styles.blocks`
- Template parts: `templateParts`
- Pattern references: `patterns`
- Style variations: `styles/*.json`

## Frontend Skill Links

- Color decisions still use Dynamic UI Color Contrast Logic.
- Spacing decisions still use Dynamic UI Spacing Rhythm Logic.
- Typography decisions still use Dynamic UI Typography Logic.
- Layout decisions still use Layout & Composition Fundamentals.
- Rendered claims still use Frontend Inspection Discipline.
