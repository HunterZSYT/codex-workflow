# WordPress Scroll Scene Verification Checklist

- [ ] Scene starts and ends at intended scroll positions.
- [ ] Pin spacing does not create blank gaps.
- [ ] Header/sticky elements do not overlap critical content.
- [ ] Page remains scrollable with keyboard.
- [ ] Mobile viewport is checked with device-style emulation.
- [ ] Console has no GSAP/ScrollTrigger/Lenis errors.
- [ ] Images/fonts have loaded before final ScrollTrigger refresh.
- [ ] WooCommerce/cart/checkout pages are excluded unless explicitly animated.
