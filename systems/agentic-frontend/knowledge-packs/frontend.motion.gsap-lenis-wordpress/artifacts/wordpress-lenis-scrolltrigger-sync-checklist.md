# WordPress Lenis / ScrollTrigger Sync Checklist

- [ ] Initialize Lenis once.
- [ ] Register ScrollTrigger.
- [ ] Wire Lenis scroll events to `ScrollTrigger.update`.
- [ ] Drive Lenis RAF from the animation ticker if using that integration.
- [ ] Refresh ScrollTrigger after images/fonts/layout shifts.
- [ ] Test anchor links and browser history behavior.
- [ ] Disable/simplify for reduced motion.
- [ ] Check mobile scroll performance before claiming smooth behavior.
