Best-practice rules:
- Use a clear column grid and consistent container edge.
- Let typography, spacing, and alignment carry the design before decoration.
- Use asymmetry deliberately, with stable alignment anchors.
- Keep color restrained and make contrast, scale, and whitespace do the work.
- Maintain strong section rhythm and avoid card clutter.

Implementation pattern:
- Establish container and column grid.
- Define type hierarchy first.
- Place focal content on clear grid lines.
- Use fewer, stronger sections with controlled whitespace.
- Verify mobile stacking preserves the intended reading order.

Anti-patterns:
- Random offset elements without grid anchors.
- Decorative blobs or gradients replacing composition.
- Too many card containers in an editorial layout.

- Inspect grid alignment, whitespace rhythm, readable type, and mobile order.
