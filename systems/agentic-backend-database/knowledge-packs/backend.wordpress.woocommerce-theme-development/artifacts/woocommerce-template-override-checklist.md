# WooCommerce Template Override Checklist

- [ ] Confirm hooks/filters cannot solve the change.
- [ ] Confirm classic vs block template path.
- [ ] Use a child theme for classic PHP overrides.
- [ ] Copy only the minimum required template.
- [ ] Preserve relative folder structure.
- [ ] Add a comment/note with WooCommerce template version if relevant.
- [ ] Check WooCommerce status for outdated templates.
- [ ] Re-test affected storefront flow.
- [ ] Keep a rollback copy or commit.
