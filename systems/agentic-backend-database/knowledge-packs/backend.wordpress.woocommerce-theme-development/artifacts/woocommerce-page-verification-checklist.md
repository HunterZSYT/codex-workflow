# WooCommerce Page Verification Checklist

- [ ] Shop/product archive renders.
- [ ] Product category/tag archives render.
- [ ] Single product renders.
- [ ] Product images/gallery work.
- [ ] Variation selectors work if variable products exist.
- [ ] Add to cart works.
- [ ] Cart totals update.
- [ ] Checkout loads in safe test/staging mode.
- [ ] Payment section is not visually or functionally broken.
- [ ] Account pages render.
- [ ] Order confirmation renders if touched.
- [ ] Logs show no relevant new PHP/WooCommerce errors.
