# WooCommerce Gallery / Carousel Reuse Scout

Before custom carousel code:

- Check WooCommerce/product gallery defaults.
- Check whether the theme already provides gallery behavior.
- Check whether a maintained plugin/library is already installed.
- Check accessibility: keyboard, focus, reduced motion, labels.
- Check product image zoom/lightbox requirements.
- Check mobile swipe behavior.
- Avoid React SPA assumptions unless the theme architecture supports React.
- Verify performance and layout shift on product pages.

If a new library is needed, use ecosystem scout and ask approval before installing.
