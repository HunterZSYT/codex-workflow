# WooCommerce Child Theme Override Map

Classic template source:
- WooCommerce plugin `templates/single-product/*.php`
- WooCommerce plugin `templates/archive-product.php`
- WooCommerce plugin `templates/cart/*.php`
- WooCommerce plugin `templates/checkout/*.php`
- WooCommerce plugin `templates/myaccount/*.php`

Child theme target:
- Child theme `woocommerce/single-product/*.php`
- Child theme `woocommerce/archive-product.php`
- Child theme `woocommerce/cart/*.php`
- Child theme `woocommerce/checkout/*.php`
- Child theme `woocommerce/myaccount/*.php`

Do not include the source `templates/` segment in the child theme path.
