# Verification

For real WooCommerce theme work:

- Check whether the project uses classic templates, block templates, or both.
- Confirm no WooCommerce plugin files were edited.
- Confirm child-theme override paths are correct.
- Check WooCommerce status/outdated template notices.
- Verify product archive.
- Verify single product.
- Verify product gallery/carousel if touched.
- Verify cart.
- Verify checkout using a safe test mode or staging environment.
- Verify account and order confirmation if touched.
- Check logs and rollback plan before live deployment.
