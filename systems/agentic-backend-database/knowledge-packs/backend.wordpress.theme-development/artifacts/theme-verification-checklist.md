# Theme Verification Checklist

- [ ] Changed PHP files pass syntax checks.
- [ ] Theme activates without fatal errors.
- [ ] `WP_DEBUG_LOG` has no relevant new errors.
- [ ] Front page renders.
- [ ] Single post/page renders.
- [ ] Archive/search/404 render if touched.
- [ ] Header/footer/template parts render.
- [ ] Enqueued assets appear once and in the expected order.
- [ ] Block theme editor preview matches front-end reasonably.
- [ ] Mobile layout is checked with real emulation when a mobile claim is made.
