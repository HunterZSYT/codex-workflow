# Manual Auth Needed

This repo does not store secrets or live auth state.

Configure manually after restore:

- Codex login/auth
- `auth.json`
- real `config.toml`
- GitHub auth
- Figma Desktop / Dev Mode auth
- Supabase/Postgres credentials
- SSH keys and SSH config
- VPS/server passwords or key passphrases
- browser cookies/profile state

Never commit these files or values to this repository.
